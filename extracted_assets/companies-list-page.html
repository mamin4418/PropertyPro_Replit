Implementation Notes:

  • The search should filter companies in real-time as the user types.
  • The filter dropdown should allow filtering by company type.
  • The view toggle should switch between grid and list views.
  • Each company card should have quick actions (View, Edit, Delete).
  • The "Add Company" button should open a form to create a new company.
  • The pagination should allow navigation between pages of companies.
  • Clicking on "View" should navigate to the company detail page.
  • Clicking on "Edit" should open the company edit form.
  • Clicking on "Delete" should show a confirmation dialog before deleting.
  • The occupancy rate should be calculated based on the company's properties and units.
